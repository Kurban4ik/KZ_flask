Этот проект - сайт Unique Changer
проект предназначен для редактирования фотографий четырьмя уникальными фильтрами
для запуска проекта нужно запустить код main.py
в blueprints хранятся блюпринты (дополнительные части) проекта
пока что тут лежит только API часть проекта
в data хранятся базовые части проекта на flask - __all_models.py для регистрации,
db_session.py код для взаимодействия с базой данных
также тут расположены две модели - photos.py и users.py
для запуска проекта нужно скачать все библиотеки из requirments.txt
reesp.py используется для теста приложения
